# CasDoc

**CasDoc** is an intelligent documentation helper tool powered by OpenAI ChatGPT, designed to automatically polish and translate documentation for software projects. Developed by the [Casbin](https://github.com/casbin) organization, CasDoc streamlines the documentation process by leveraging AI to enhance content quality and accessibility.

## What is CasDoc?

CasDoc is a command-line tool written in Go that helps developers and technical writers:

- **Polish Documentation**: Automatically improve the quality, clarity, and readability of existing documentation
- **Translate Documentation**: Convert documentation into multiple languages (Chinese, French, German, Korean, Russian, Japanese)
- **Batch Processing**: Handle entire documentation directories efficiently
- **AI-Powered**: Utilize OpenAI's ChatGPT for intelligent content enhancement
