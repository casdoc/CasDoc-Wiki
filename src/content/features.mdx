### Translation Modes
```bash
# Translate to specific languages
casdoc --translate --languages zh,fr,de

# Translate to all supported languages
casdoc --translate --all-languages

# Preserve original files
casdoc --translate --preserve-original
```

## üìÅ File Management

Comprehensive file handling capabilities for various documentation structures.

### File Discovery
- **Recursive Scanning**: Automatically finds all documentation files
- **Smart Filtering**: Processes only relevant file types
- **Pattern Matching**: Supports include/exclude patterns
- **Size Limits**: Configurable file size restrictions

### Supported Formats
- **Markdown**: `.md` files (primary format)
- **MDX**: `.mdx` files with React components
- **Text Files**: `.txt` files for simple documentation
- **Custom Extensions**: Configurable file type support

### File Operations
```bash
# Process specific file types
casdoc --include "*.md,*.mdx"

# Exclude certain patterns
casdoc --exclude "**/node_modules/**,**/vendor/**"

# Set file size limits
casdoc --max-file-size 1MB
```

## üîÑ Backup and Recovery

Protect your documentation with built-in backup and recovery features.

### Automatic Backups
- **Pre-Processing Backups**: Automatic backup before any modifications
- **Configurable Backup Location**: Choose where backups are stored
- **Backup Naming**: Timestamp-based backup file naming
- **Cleanup Options**: Automatic old backup removal

### Recovery Options
```bash
# Enable backups (default)
casdoc --create-backups

# Custom backup suffix
casdoc --backup-suffix .casdoc-backup

# Restore from backup
casdoc --restore docs/README.md.bak
```

### Version Control Integration
- **Git Integration**: Works seamlessly with Git repositories
- **Commit Hooks**: Can be integrated into Git pre-commit hooks
- **Diff-Friendly**: Changes are easy to review in version control

## üîß Configuration Flexibility

Extensive configuration options for different workflows and requirements.

### Environment Variables
```bash
# Core configuration
export CASDOC_AUTHTOKEN="your-api-key"
export CASDOC_REPOPATH="/path/to/docs"
export CASDOC_MODEL="gpt-3.5-turbo"
export CASDOC_TEMPERATURE="0.3"
```

### Configuration Files
```toml
# config.toml
[general]
api_timeout = 30
max_retries = 3

[processing]
batch_size = 5
concurrent_workers = 2

[translation]
default_languages = ["zh", "fr", "de"]
preserve_code = true
```

### Command-Line Overrides
```bash
# Override any configuration at runtime
casdoc --model gpt-4 --temperature 0.1 --batch-size 10
```
